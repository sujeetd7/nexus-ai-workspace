# Branch Naming Convention

feature/<module>-<feature>

Examples:

feature/auth-login
feature/chat-streaming
feature/rag-upload
feature/agent-runtime
feature/mcp-github

bugfix/<issue>

Examples:

bugfix/auth-token
bugfix/chat-scroll

hotfix/<issue>

Examples:

hotfix/prod-login
hotfix/security

release/<version>

Examples:

release/v1.0.0
release/v1.1.0
